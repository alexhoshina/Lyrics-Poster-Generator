# 歌词海报生成器 (Lyrics Poster Generator)

歌词海报生成器是一个基于Python的脚本,可以生成一张简单的歌词海报

## 项目介绍

歌词海报生成器使用Python编写，借助PIL生成一张简单的歌词海报。歌词海报生成器可以指定背景图片、歌曲标题、歌词内容和字体。  
这是我个人的一个练手项目，也是第一次尝试完成一个相对学校大作业来说更加完整的项目。
